# How to Install "ChatGPTArchiver"
## A Step-by-Step Guide for Corporate Users

**Target Audience:** Non-technical staff | **Time Required:** 2 minutes

---

### 1. Prepare the Files
1.  **Download** the **[chatgpt-archiver.zip](../dist/chatgpt-archiver.zip)** file from the `dist` folder of this repository.
2.  **Unzip** the file.
    *   *Right-click* the zip file and select **"Extract All..."** or **"Unzip"**.
    *   Remember where you saved this folder (e.g., your "Downloads" or "Documents" folder).

### 2. Open Chrome Extensions Settings
1.  Open your **Google Chrome** browser.
2.  In the address bar at the very top, type this exact text and press **Enter**:
    `chrome://extensions`
    *(Alternatively: Click the puzzle piece icon ðŸ§© in the top right -> Manage Extensions)*

### 3. Enable "Developer Mode"
*Don't worry, this is safe! It simply allows you to install internal company tools.*

1.  Look at the **top right corner** of the Extensions page.
2.  Find the switch labeled **"Developer mode"**.
3.  Click it to toggle it **ON** (the switch usually turns blue).

### 4. Load the Extension
1.  Three new buttons will appear in the top left. Click the first one: **"Load unpacked"**.
2.  A file browser window will open.
3.  Navigate to the **folder you unzipped** in Step 1.
    *   *Crucial:* Select the *folder itself*, not a file inside it.
4.  Click **"Select"** (or "Open").

### 5. Success!
You should now see "ChatGPTArchiver" in your list of extensions.

---

### How to Use It
1.  Go to [ChatGPT (chat.openai.com)](https://chat.openai.com).
2.  Click the **Extensions Puzzle Piece (ðŸ§©)** in the top right of Chrome.
3.  Click **ChatGPTArchiver**.
4.  Choose **PDF** or **Word** (Text) and click **Export**.

### FAQ
*   **Q: Is my data safe?**
    *   A: Yes. This tool runs 100% on your computer. No conversation data is sent to any outside server.
*   **Q: The "Batch Export" is slow.**
    *   A: This is normal. To prevent errors, the tool processes one conversation at a time, just like a human would.
